How it works

1. Your design – File upload

Our machines need to know where to cut, and where and how to bend, if required.

The best file format is an AutoCAD drawing file (dwg, dwt, dxf, dws, dwf, dwfx, dxb). Show bend positions, angle and bend radius if bends are required.

If you can’t produce an AutoCAD file, we can work with:

  • A picture with measurements
  • A sketch with measurements
  • A different CAD format

Specify material type, thickness and surface finish. Unsure? Ask us.

2. Final design & quote

We send a final design for approval plus a quote with estimated delivery.

We may propose alternatives that improve manufacturability or cost.

Quote depends on material, weight, cut length, bends, tolerances, design assistance, and special processes.

3. Payment

A secure payment link is sent. Pay via Debit/Credit Card or Ozow Instant EFT. Manufacture begins once payment clears.

4. Manufacture and shipping

We keep you informed and send tracking details when shipped.

5. Support and returns

We help with questions and assembly. If we made an error, we replace the part.
